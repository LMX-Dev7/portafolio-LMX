# Portafolio

Estructura escalable y profesional (2026)

- **assets/**: Recursos estáticos (imágenes, CSS, JS)
- **components/**: Componentes reutilizables (navbar, footer, hero, etc.)
- **projects/**: Proyectos
- **about/**: Información personal/profesional
- **index.html**: Entrada principal del portafolio

Cada sección importante está separada en su propio archivo para facilitar el mantenimiento y la escalabilidad.

**Incluye los componentes en index.html usando frameworks, includes o JS según tu stack.**
